jQuery UI demo

This html file uses code provided in by the jQuery UI Download Builder. From there, I downloaded a zip archive that provided me with js, css, images,and an index.html that shows all the UI widgets included in the library. The css corresponds the "dark" theme I chose on the download page.

I removed the files I didn't need for this example (extra CSS, images) and adapted the code they provided in the index.html example (not included here). I also decided to use remote URLs in the script tags for jquery.js and jquery-ui.js, rather than include these files locally.

I did not keep the same directory structure as in the downloaded zip file, but to respect the authors' copyright license, I kept LICENSE.txt file and AUTHORS.txt file. There's also a package.json file, which we don't need in this case, but I kept it to show you info about jQuery UI and its dependencies.
