# How to use timers
## setTimeout
Sets the number of milliseconds, after which the function will be called.

* a function
* number of ms    
`setTimeout(foo,1000);`

## clearInterval
Clears the timer
```
x = setInterval(foo, 1000);
clearInterval(x);
```
